# ElrsTransmitter
A little transmitter in elrs with a esp32 and a sx1280
# Why did I make this transmitter
-Easy to use and easy to change   
-Can use every esp32 or esp8266 receiver with espnow i case of a broken elrs receiver  
# Protocols 
It can use espnow and elrs
# Components
-Esp32 devkit C (30pin)  
-E28-2G4M27SX (SX1280)  
-2 gimbal  
-4 switch 3 position  
-a screen   
-a button for the configuration  
-a 4900mah battery  
-a 1s lipo charger   
-a 3.3v regulator  
# Where is the code ?
Elrs protocol is very difficult and not my level so I use the code of ExpressLRS here is his github:
https://github.com/ExpressLRS/ExpressLRS/tree/master
